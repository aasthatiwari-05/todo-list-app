# To-Do List App 📝

A simple and elegant web-based to-do list application built with HTML, CSS, and JavaScript. This app helps you manage your daily tasks efficiently with a clean, user-friendly interface.

![To-Do List](https://img.shields.io/badge/Version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

## ✨ Features

- ➕ **Add Tasks**: Easily add new tasks to your list
- ✅ **Mark as Complete**: Click on tasks to mark them as completed
- ❌ **Delete Tasks**: Remove tasks by clicking the delete button (×)
- 💾 **Auto-Save**: All tasks are automatically saved in browser's local storage
- 🎨 **Beautiful UI**: Modern gradient design with smooth user experience
- 📱 **Responsive**: Works perfectly on all screen sizes

## 💻 Usage

1. **Adding a Task**: Type your task in the input box and click "Add"
2. **Completing a Task**: Click on any task to mark it as complete
3. **Deleting a Task**: Click the × button on the right side of any task
4. **Persistence**: Your tasks are automatically saved and will reappear when you return
